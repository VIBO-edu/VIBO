## 📖 About VIBO

VIBO is the ultimate collaboration platform for college students to **share study materials**, **chat with friends**, **express themselves**, and **grow together**. Think of it as **Instagram + WhatsApp + Study Portal** for college students.

### 🎯 Purpose
- **Study**: Share notes, assignments, and past papers
- **Fun**: Connect with friends and share moments
- **Confession**: Share thoughts anonymously with admin approval

---

## ✨ Features

### 📚 Study Materials
- Share notes, assignments, and past papers
- Like, comment, and download resources
- Organized by Faculty → Class → Subject

### 💬 Real-time Chat
- Individual & Group messaging
- File sharing in chats
- Cross-class chat (with admin approval)
- Read receipts & typing indicators

### 📝 Social Feed (Instagram-style)
- **Study Posts** - Share academic content
- **Fun Posts** - Socialize with friends
- **Confessions** - Anonymous sharing (admin approved)

### 💭 Confession Feature
- Fully anonymous
- Admin approval required
- Expires in 24 hours
- "Confession of the Day" selected randomly

### 👥 Friend System
- Smart suggestions based on class/section
- Send/accept/reject requests
- Block & mute options

### 📁 File Sharing
- Upload documents, images, videos
- 20MB limit, 1GB storage per user
- Track downloads

### 🔍 Smart Search
- Search users, posts, materials, files
- Advanced filters (faculty, class, type)

### 👑 Admin Dashboard
- Super Admin + Faculty Admins
- Content moderation tools
- Analytics & reporting
